# 幸存者 · Survivor

一款类似 **吸血鬼幸存者** 和 **土豆兄弟** 的 Roguelike 生存游戏。

## 快速开始

直接用浏览器打开 `index.html` 即可游玩，无需安装任何依赖。

或使用本地服务器（推荐）：

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

然后访问 http://localhost:8080

## 操作

| 按键 | 功能 |
|------|------|
| W/A/S/D 或 方向键 | 移动 |
| 空格 | 暂停/继续 |
| 鼠标 | 选择升级/商店物品 |

## 游戏特色

### 角色选择
- **战士** — 高生命，起始武器：鞭刃
- **法师** — 高伤害，起始武器：魔弹
- **游侠** — 高速度，起始武器：飞刀

### 武器系统（自动攻击）
- 🌀 **鞭刃** — 横扫前方敌人
- ✨ **魔弹** — 追踪最近的敌人
- 🗡️ **飞刀** — 快速穿透
- ⚡ **闪电链** — 连锁攻击
- 🔮 **护体球** — 环绕伤害
- 🔥 **火球** — 爆炸范围伤害

### 升级 & 商店
- 击杀敌人获得经验宝石，升级后三选一强化
- 每波结束（30秒）进入商店，用金币购买属性
- 存活 5 分钟即胜利，第 10 波出现 Boss

## 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES Modules)
- 无外部依赖
